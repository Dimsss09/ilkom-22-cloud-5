Panduan Penggunaan Aplikasi

Ikuti langkah-langkah berikut untuk menggunakan aplikasi ini:

  1. Login menggunakan akun yang telah diberikan oleh admin.
  2. Pilih menu sesuai kebutuhan (contoh: Upload File, Input Data, dll).
  3. Isi form dengan data yang benar dan lengkap.
  4. Klik tombol "Simpan" untuk menyimpan data Anda.
  5. Gunakan tombol "Logout" untuk keluar dengan aman dari aplikasi.
